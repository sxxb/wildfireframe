What does this page do?

This is the reporting page - people fill in information to report a hazard near them

What do we need for this page to work?

There are a few things we need:

  • Form inputs using pre-saved information OR new inputs, including complex inputs such as a map location selector.
  • The form information will be pulled from a table in a db, each user will probably have a table? - Look into database structure best professional practices
  • The form submission will insert a new entry into a table in a db (using PHP?) - the same DB reports are pulled from on the status page
  • The form submission will lead to another report page, prompting for more information if the user is in a position to provide it.
